# Application: [Headspace] 
- Most important difference between pipelines: [The manual pipeline allows for more focused and well thought out groupings and requirements, while the automated pipeline is a lot more streamlined]
- Most useful pipeline: [The hybrid pipeline is the most useful because it combines the best aspects of both manual and automatic. A baseline can be generated automatically and then revised and refined manually, which allows for the same level of depth while saving a lot of time.] 
- Most surprising finding: [The most surprising finding was while studying the metrics, the hybrid metric has the highest traceability ratio, while also having the lowest review coverage ratio.]
- Observed weakness in the automated pipeline: [The biggest weakness of the automated pipeline is that pipeline outputs often lack depth. It is good for sorting through large sets of data, but the manual pipeline is more about creating realistic personas and useful requirements.]
