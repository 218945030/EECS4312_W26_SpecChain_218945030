# Requirement ID: FR1 

- Description: [The system shall allow free users to access 1/4 of the apps content.]
- Source Persona: [Low Income User]
- Traceability: [Derived from review group G1]
- Acceptance Criteria: [Given the user accesses the free version of the app, When the user selects a feature to access Then the application will provide them with 1/4 of the options of that feature.]

# Requirement ID: FR2 

- Description: [The system shall not crash during guided meditations.]
- Source Persona: [Frequent Meditator]
- Traceability: [Derived from review group G2]
- Acceptance Criteria: [Given the user is listening to a guided meditation, When listening to the meditation Then the application will not crash while the meditation is playing.]

# Requirement ID: FR3 

- Description: [The system shall load options in less than 1 minute.]
- Source Persona: [Frequent Meditator]
- Traceability: [Derived from review group G2]
- Acceptance Criteria: [Given the user is choosing a guided meditation, When accessing the list of guided meditations Then the application will load all available options in under 1 minute.]
